# NavigationEvent object

An object that represents an event fired when navigation is initiated.

## Properties
    
### uri

The Uniform Resource Identifier (URI) of the target.

This property is read-only.

```js
var uri = NavigationEvent.uri;
```

#### Property value
Type: **DOMString**
